Bewegung mit konstanter Beschleunigung
Mathematischer Anhang

Ein im Idealfall punktförmiger Körper bewegt sich geradlinig mit konstanter Beschleunigung. Die momentane Position des Körpers zur Zeit t wird durch die Koordinate x beschrieben. Gesucht ist die Abhängigkeit dieser Koordinate von der Zeit.

t .... Zeit
s .... zurückgelegte Strecke
a .... Beschleunigung
v0 ... Anfangsgeschwindigkeit

Wenn die Anfangsposition des Körpers nicht mit dem Nullpunkt übereinstimmt, muss das berücksichtigt werden.

t .... Zeit
x .... Koordinate (momentane Position)
a .... Beschleunigung
v0 ... Geschwindigkeit zur Zeit t = 0 (Anfangsgeschwindigkeit)
x0 ... Koordinate zur Zeit t = 0 (Anfangsposition)

t .... Zeit
v .... (Momentan-)Geschwindigkeit
a .... Beschleunigung
v0 ... Geschwindigkeit zur Zeit t = 0 (Anfangsgeschwindigkeit)
